---

## **Reporting a Vulnerability**

If you discover a security vulnerability in this project, please follow these guidelines:

1. **Contact**: Send a detailed report to **[amrmohamed18032010@gmail.com]**.
2. **Expected Response Time**: You can expect a confirmation within **48 hours**.  
3. **Updates**: You will receive progress updates **as the issue is assessed and resolved**.  
4. **Resolution**:  
   - If accepted, a patch will be released and you will be credited if you choose.  
   - If declined, a clear explanation will be provided.  

> :warning: Do **not** post vulnerabilities publicly until they have been resolved to prevent exploitation.

---

## **Best Practices for Reporters**

- Include clear steps to reproduce the issue.  
- Provide environment details (OS, version, dependencies).  
- Include relevant code snippets or screenshots if possible.  

Thank you for helping keep this project **safe and secure**.
